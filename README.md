# PHOTOCILA — Galeria

Galeria de fotografias documentais de Priscila Corrêa (@photocila).

## Como gerir

Tudo o que você precisa editar está em **`src/colecao.js`**:

- Seu número de WhatsApp (campo `whatsapp`)
- Tamanhos, molduras e preços
- A lista de fotografias

Para adicionar uma foto: suba a imagem no Cloudinary, copie o link,
e adicione um novo bloco em `COLECAO` dentro de `src/colecao.js`.

## Publicação

Este projeto é publicado pela Vercel, conectada a este repositório.
Cada alteração salva aqui é republicada automaticamente.
